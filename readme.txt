A simple PDF output plugin built for Wordpress.

== Description ==

A simple plugin to output PDF’s attached to posts.

== Installation ==

Paste the following shortcode into your template where you’d like to use the plugin: 

<?php echo do_shortcode(‘[pdf_shortcode]’); ?>
